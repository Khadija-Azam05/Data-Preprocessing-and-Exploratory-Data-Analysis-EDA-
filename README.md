# Data-Preprocessing-and-Exploratory-Data-Analysis-EDA

This project mainly focuses on **Data Preprocessing** and **Exploratory Data Analysis (EDA)** in **Data Science / Machine Learning** using employee datasets.

## 📌 Project Overview

The project demonstrates essential data science techniques including data cleaning, missing value treatment, outlier detection, feature engineering, visualization, aggregation, encoding, scaling, and dataset integration using Python libraries such as **Pandas, NumPy**.

---

# 📂 Topics Covered

## → Data Cleaning & Data Wrangling
- Loading datasets
- Checking dataset shape and data types
- Handling duplicates
- Fixing inconsistent categorical values
- Filtering and slicing data
- Standardizing categorical entries


## → Missing Value & Outlier Treatment
- Handling missing values using:
  - Mean
  - Median
  - Mode
  - "Unknown" category
- Outlier detection using:
  - IQR Method
  - Z-Score Method
- Outlier treatment:
  - Removing outliers
  - Capping outliers


## → Exploratory Data Analysis (EDA) & Feature Transformation
- Descriptive statistics
- Histograms
- Boxplots
- KDE plots
- Skewness detection
- Feature transformations:
  - Log Transformation
  - Box-Cox Transformation
  - Yeo-Johnson Transformation


## → Data Aggregation & Business Analytics
- GroupBy operations
- Aggregation functions
- Department-wise analysis
- Gender-wise performance analysis
- Pivot tables for business insights


## → Feature Engineering & Data Scaling
- Label Encoding
- One-Hot Encoding
- Min-Max Scaling
- Standardization (Z-score)
- Robust Scaling


## → Data Integration & Comparative Analysis
- Dataset merging/joining
- Creating derived features
- Training hour analysis
- Comparative salary analysis
- Certification-based comparisons

# 🛠️ Technologies Used

- Python

- NumPy
